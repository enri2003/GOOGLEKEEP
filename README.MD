# GOOGLEKEEP

Aplicacion full stack inspirada en Google Keep, construida con:

- `Frontend`: Angular + PrimeNG
- `Backend`: NestJS + TypeORM + JWT
- `Base de datos`: PostgreSQL

## Funcionalidades

- Registro e inicio de sesion
- Perfil del usuario autenticado
- Cierre de sesion seguro
- CRUD de notas
- Recordatorios, archivo y papelera
- Proteccion de rutas con JWT

## Estructura

- `frontend/`: interfaz de usuario
- `backend/`: API y autenticacion
- `docker-compose.yml`: apoyo para entorno local

## Ejecutar el proyecto

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```
